## 安装并使用ollama

- [ollama homepage](https://ollama.com/)
- download and install [ollama CLI](https://ollama.ai/download)

## 开始
- 克隆代码
- 运行npm install安装依赖，运行npm run dev运行网页
- 根据不同平台执行不同的cmd脚本，在本地运行ollama服务

```
git clone https://github.com/GGMGG/ollamaWeb.git
npm install
npm run dev
```

```
ollama serve
```
